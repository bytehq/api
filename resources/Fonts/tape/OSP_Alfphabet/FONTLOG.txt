
FONTLOG for Alfphabet-IV
-------------------
This file provides detailed information on the Alfphabet-IV font software.
This information should be distributed along with the Alfphabet-IV fonts
and any derivative works.

Basic Font Information
--------------------------
Copyright (c) 2007-2010, OSP (http://ospublish.constantvzw.org)
The Alfphabet family is based on the Belgian road signage (1945-1975, originally from 3M system working for the Marshall plan after the end of the war).
Alfphabet IV was redrawn by Pierre Huyghebaert and Ludi at Speculoos studio in 2007.

Alfphabet-IV currently provides the following Unicode coverage:
Basic Latin: 95/95 
Latin-1 Supplement: 60/96
